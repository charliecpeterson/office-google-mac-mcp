# google-mcp

**Work in progress.** An MCP server to work live inside open Google
Docs/Sheets/Slides via the Google Workspace APIs — the cross-platform sibling of
the macOS `office-mcp`. Edits go through the official REST APIs and show up live in
the open browser tab via Google's realtime sync.

See [PLAN.md](PLAN.md) for the design. Sheets is built first (cleanest API).
